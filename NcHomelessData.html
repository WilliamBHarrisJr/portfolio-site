    About This Project

    Python

    SQL

    Python data pipeline to analyze NC Point-in-Time homeless data for 2007-2024, cleaning and merging data with population estimates and correcting units. Visualizes trends using Matplotlib and predicts 2025 homelessness with a simplified Random Forest model, uploads data to PostgrSQL database. SQL table downloads to interactive Power BI dashboard with line charts, a dual-axis chart, a metrics table, and a 2025 prediction card.
